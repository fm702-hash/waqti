# ⏱️ وقتي — تطبيق PWA

> أنجز مهامك • اكسب وقتك

---

## 🚀 تشغيل المشروع محلياً

```bash
# 1. تثبيت المكتبات
npm install

# 2. تشغيل للتطوير
npm run dev

# 3. بناء النسخة النهائية
npm run build

# 4. معاينة النسخة النهائية
npm run preview
```

---

## ☁️ الرفع على Vercel (مجاناً)

### الطريقة الأولى — عبر GitHub (الأسهل)

1. ارفع المشروع على GitHub
2. اذهب إلى https://vercel.com
3. اضغط "Add New Project"
4. اختر الـ Repository
5. Vercel يكتشف Vite تلقائياً ويضبط كل شيء
6. اضغط Deploy ✅

### الطريقة الثانية — CLI مباشرة

```bash
npm install -g vercel
vercel login
vercel --prod
```

## 📱 تثبيت التطبيق على الجهاز

### iPhone / iPad
1. افتح الرابط في **Safari** (ليس Chrome)
2. اضغط زر المشاركة 🔗
3. اختر **"إضافة إلى الشاشة الرئيسية"**
4. اضغط إضافة ✅

### Android
1. افتح الرابط في **Chrome**
2. سيظهر بانر تلقائي "تثبيت التطبيق"
3. أو اضغط القائمة ⋮ ← "إضافة إلى الشاشة الرئيسية"

## 🗂️ هيكل المشروع

```
waqti/
├── index.html              ← نقطة الدخول
├── vite.config.js          ← إعدادات Vite + PWA
├── package.json
├── public/
│   └── icons/
│       ├── icon-192.png    ← أيقونة التطبيق (أضفها يدوياً)
│       └── icon-512.png    ← أيقونة كبيرة (أضفها يدوياً)
└── src/
    ├── main.jsx            ← نقطة بدء React
    ├── App.jsx             ← التطبيق الكامل
    ├── data.js             ← قاعدة بيانات المهام والأسئلة
    └── storage.js          ← التخزين المحلي (offline)
```

## ✨ الميزات

- ✅ يعمل بدون إنترنت (Offline)
- ✅ يُثبَّت على الشاشة الرئيسية
- ✅ واجهة الطفل + ولي الأمر
- ✅ كويزات من المنهج السعودي
- ✅ نظام الكود لكسب وقت الشاشة
- ✅ الكود ينتهي تلقائياً بعد 120 ثانية
- ✅ حفظ البيانات يومياً (يُعاد ضبطها يومياً)

## 🔧 إضافة المزيد من الأسئلة

افتح `src/data.js` وأضف أسئلتك في مصفوفة `questions`:

```js
{
  id: 6, icon: "📐", title: "كويز الهندسة",
  subject: "رياضيات", grade: "الصف الخامس",
  reward: 25, color: "#FF9F43",
  questions: [
    { q: "كم زاوية المثلث؟", opts: ["2","3","4","5"], ans: 1 },
  ]
}
```
